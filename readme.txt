== Description ==

This plugin replaces the WordPress 3.6+ post format <abbr title="User Interface">UI</abbr> with the pre-3.6 meta box for selecting a post format.

== Installation ==

1. Upload `post-format-meta-box` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= Why was this plugin created? =

I prefer the old post format meta box over the new post format UI.  I figured some other folks might like this too.

= How do I use it? =

Install and activate it.  That's it!
